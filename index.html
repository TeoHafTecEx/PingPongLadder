🏓 TT Ladder Setup

One-time GitHub connection
1Create data.json in your repo
In your GitHub repo root, create a file called data.json with exactly this content: 
{"leagueName":"Complex Solutions","players":[],"matches":[],"nextMatchId":1}
2Create a Fine-grained Personal Access Token
Go to GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens.

Set Repository access to your ladder repo only.
Under Repository permissions, set Contents → Read and Write.

Copy the generated token — you won't see it again.
3Enter your details below
Each person who logs matches needs to do this setup once on their device. The token is stored in your browser only and never leaves it.
⚠️ Stored in your browser only (localStorage). Never transmitted except to GitHub's API.
